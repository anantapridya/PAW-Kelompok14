- Clone this project
- Install **Express JS** , **Ejs**, **MongoDB**, **Mongoose**, **Cors**, and **dotenv** with **npm install**
- Create **.env** file on **Server** folder
- Install package.json with command
  ```bash
  npm i
  ```
- Enter cd Server
- Run with command
  ```bash
  npm run dev
  ```
- Open your browser then go to localhost:9000
- Try API on Postman
