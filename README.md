### Panduan

- [Download](https://nodejs.org/en/download/) and install **Node.js** version `16.15` or higher.
- Clone this project
- Install **Express JS** , **Ejs**, **MongoDB** and **Mongoose** with **npm install**
- Start the program with command **npm start**
- Open **localhost:3000**
